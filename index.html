Notificações

Verificando…

Ao abrir o app, você recebe uma notificação se houver contas vencendo nos próximos 3 dias ou atrasadas. O ícone do app mostra o número de pendências.

Cartão de crédito

Nenhum cartão configurado. Configure para lançar gastos e contas no crédito — a fatura aparece automaticamente na aba Contas.

Cotações (brapi.dev)

Crie uma conta gratuita em brapi.dev, gere seu token e cole aqui. O botão "Atualizar cotações" na aba Aportes usa esse token (fica salvo só neste aparelho). Plano gratuito: ~38 atualizações/dia para 13 papéis.

Backup dos dados

Tudo fica salvo apenas neste aparelho. Exporte um arquivo JSON de tempos em tempos.

Fase 2 — push com o app fechado

Este app já tem o receptor de Web Push pronto no service worker. Quando você conectar um backend (FastAPI + pywebpush), as notificações chegarão mesmo com o app fechado.
